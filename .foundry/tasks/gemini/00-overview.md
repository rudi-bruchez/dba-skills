# DBA Skills Project Overview

## Goal
The goal is to create a set of specialized agent skills for Database Administration (DBA) tasks, focusing on Microsoft SQL Server and PostgreSQL. These skills will follow the specification at https://agentskills.io/specification.

## Project Structure
The skills will be organized into two main categories:
1. **Microsoft SQL Server Skills**
2. **PostgreSQL Skills**

Each skill will be a standalone directory containing a `SKILL.md` file and optional `scripts/` or `references/` directories.

## Roadmap

- Define the roadmap and specific tasks for each skill.
- Create task files in `gemini-tasks/`.
