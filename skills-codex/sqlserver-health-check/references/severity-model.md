# Severity and Confidence Model

## Severity
- `critical`: current outage, data loss risk, or severe SLA breach.
- `high`: major degradation with clear production impact.
- `medium`: notable issue with moderate risk if unchanged.
- `low`: optimization or hygiene improvement.

## Confidence
- `high`: consistent evidence across multiple sources.
- `medium`: partial evidence with some unknowns.
- `low`: limited visibility or missing permissions.

## Required Finding Format
1. Symptom
2. Evidence
3. Likely Cause
4. Next Action
5. Risk if Deferred
