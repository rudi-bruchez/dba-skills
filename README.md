# DBA Skills for Agentic Coding

This repository contains specialized knowledge and skills designed for AI agents to perform database administration (DBA) tasks. The skills provided here enable intelligent agents to assist with common DBA operations, troubleshooting, and best practices primarily focused on **SQL Server** and **PostgreSQL**.

## Purpose

This repository serves as a knowledge base for agentic coding systems, providing AI agents with:

- **Database Administration Expertise**: Comprehensive skills for managing and maintaining database systems
- **Platform-Specific Knowledge**: Detailed information about SQL Server and PostgreSQL
- **Best Practices**: Industry-standard approaches to database management
- **Troubleshooting Guides**: Common issues and their solutions
- **Performance Optimization**: Techniques for improving database performance
- **Security Guidelines**: Best practices for database security and access control

## Supported Database Platforms

### Primary Focus

1. **Microsoft SQL Server**
   - Database administration and maintenance
   - Query optimization and performance tuning
   - Backup and recovery strategies
   - Security and compliance
   - High availability and disaster recovery

2. **PostgreSQL**
   - Database administration and configuration
   - Query optimization and indexing strategies
   - Backup and recovery procedures
   - Replication and high availability
   - Extensions and advanced features

## Use Cases for AI Agents

AI agents equipped with these skills can assist with:

- Database health checks and diagnostics
- Query performance analysis and optimization
- Backup and restore operations guidance
- Security audit and recommendations
- Index management and optimization
- Database migration planning
- Troubleshooting common database issues
- Generating SQL scripts for administrative tasks
- Explaining complex database concepts
- Providing platform-specific best practices
